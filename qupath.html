Installation Notes

It is possible to install QuPath on any plataform. You need to download the software from the Official website and follow the Installation instructions . After completing the instructions, open the program and try to open an image to make sure QuPath is installed properly.
Should you have any issues with the installation, please get in touch.

Goals of this QuPath tutorial

After completing this tutorial, the participants should be able to:

  • Load images, create projects, get familiar with the different concepts of QuPath
  • rename channels, create classes and make annotations
  • Training a Pixel Classifier
  • Run the cell detection algorithm
  • Training a cell classifier
  • Do simple scripts
  • Export the relevant data
