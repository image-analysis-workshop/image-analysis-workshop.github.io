Image J and an Introduction to Image Analysis

Installation

Fiji is just ImageJ ("Fiji"). This is a "batteries included" version of ImageJ which includes a collection of key plugins installed from the start.

Fiji can be downloaded here .

Please download, extract the files and install Fiji to your User space (the Desktop is a suggested) rather than the "Program Files" or other system-wide directories.

What is an image?

Digital images are composed of pixels. Each one of these is the visual representation of a numerical value.

Exercise 1:

  • Open Fiji
  • Go to the search bar and type "Boats"
  • Alternatively follow File > Open Samples > Boats
  • You'll see a picture of some boats
  • Hover your cursor over the different regions of the image and watch the values being displayed along the base of the toolbar.
  • This is information refers to the individual pixel you are over: xy coordinates and value

    • The image frame also displays information about the file:
  • Size in pixels
  • Bit-depth (8-bit)
  • File size
What is bit depth?
  • The range in which the values in your image exist
  • An 8-bit image as 2^8 possible values for each pixel. This is a range from 0-255.
    • It is important to think about bit depth when capturing and analysing images. You want to capture as much of the variation in intensities across your subject as possible. This means having your brightest pixels close to (but not at) the limit of your range (here 255).
